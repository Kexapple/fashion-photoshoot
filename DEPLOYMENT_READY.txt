STEP 4: Deploy Backend to Railway (10 minutes)

1. Go to: https://railway.app (sign up free)
2. Create new project
3. Import from GitHub, select fashion-photoshoot repo
4. Set root directory: backend
5. Add environment variables:
   - FIREBASE_PROJECT_ID = fashion-photoshoot-studio
   - FIREBASE_CREDENTIALS_PATH = ./firebase-credentials.json
   - NANO_BANANA_API_KEY = (optional)
   - PYTHONENV = production

6. Click Deploy
7. Copy Railway URL (e.g., project-xxx.railway.app)
